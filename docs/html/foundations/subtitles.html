Subtitles¶

class scinoephile.SubtitleSeries[source]¶

Bases: scinoephile.Base, pysubs2.ssafile.SSAFile

A series of subtitles

Extension of pysubs2’s SSAFile with additional features. Includes code for loading to and saving from hdf5. While these are part of separate classes in pysubs.SSAFile, this separation is not really beneficial here.
