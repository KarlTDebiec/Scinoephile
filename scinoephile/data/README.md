# Scinoephile Data

## Yue

The `pycantonese` HKCanCor data is a CHAT-format version of the Hong Kong
Cantonese Corpus compiled by Kang Kwong Luke from conversational Hong Kong
Cantonese recordings. `pycantonese` notes that its bundled HKCanCor data has
been substantially modified from the upstream source in data format. The
current upstream HKCanCor source referenced by `pycantonese` is:

<https://github.com/fcbond/hkcancor>

If this corpus data is used, `pycantonese` requests citation of:

Luke, K. K., and May L.Y. Wong. 2015. "The Hong Kong Cantonese Corpus: Design
and Uses." *Journal of Chinese Linguistics Monograph Series* 25: 312-333.
